Welcome to Darkie Docs. Here you will find information about Darkie Bot. On left side, you can navigate through docs to see their contents.

Introduction

Darkie Bot is powerful discord bot used for moderating your server & more. It can be used for:

  • moderating your server
  • greeting your members
  • preventing unwanted users from joining
  • communicating with friends through 4fun commands
  • assigning roles through the autorole system
  • getting server information
  • verifying users with verification system

    • To make these features work, check commands!

    Commands

    Darkie Bot has a lot of features, that can be customized. Use commands to change them as you like!

    Moderation

    Bans specific user.
    Syntax: /ban <user> <reason>

    Kicks specific user.
    Syntax: /kick <user> <reason>

    Shows list of banned users.
    Syntax: /banlist

    Mutes specific user.
    Syntax: /kick <user> <time> <reason>

    Removes mute of specific user.
    Syntax: /unmute <user>

    Warns specific user.
    Syntax: /warn <user> <reason>

    Cleares messages from channel.
    Syntax: /clear <user> <amount> <channel>

    4Fun

    Sends reaction GIF.
    List of reactions: kiss, punch, hug, slap, tickle, stare, happy, sad, cheers, bite, pat.
    Syntax: /stare <user>

    Rolls X to Y number.
    Syntax: /roll <x> <y>

    Answers a question.
    Syntax: /8ball <question>

    Sets suggestion channel.
    Syntax: /set 4fun suggestions <channel>

    Information

    Shows server information.
    Syntax: /server

    Shows server roles.
    Syntax: /roles

    Shows specific user's avatar.
    Syntax: /avatar <user>

    Shows help message.
    Syntax: /help

    Shows bot's current ping.
    Syntax: /ping

    Shows specific user's information.
    Syntax: /user <user>

    Welcome and Leave System

    Sets a message that appears when a member joins your server.
    Syntax: /set welcome message <channel> <message> <embed>
    To make it a embed, type true in embed field.
    Color field supports only hex colors.
    Syntax: /set welcome embed <title> <color>
    Removing welcome messages by typing remove command.
    Syntax: /remove welcome

    Sets a message that appears when a member leaves your server.
    Syntax: /set leave message <channel> <message> <embed>
    To make it a embed, type true in embed field.
    Color field supports only hex colors.
    Syntax: /set leave embed <title> <color>
    Removing leave messages by typing remove command.
    Syntax: /remove leave

    Autorole System

    Adds specific role to autorole system.
    Syntax: /autorole add <role>
    Removes specific role from autorole system.
    Syntax: /autorole remove <role>

    Shows autorole system roles list.
    Syntax: /autorole list

    Automod System

    Adds specific admin role.
    Syntax: /automod adminroles add
    Removes specific admin role.
    Syntax: /automod adminroles remove
    Shows admin roles list.
    Syntax: /automod adminroles list

    Adds channel to ignoring list.
    Syntax: /setmod ignore_channels add
    Removes channel from ignoring list.
    Syntax: /setmod ignore_channels remove
    Shows ignored channel list.
    Syntax: /setmod ignore_channels remove

    Verification System

    Creates verification system.
    Syntax: /automod verify create <message> <role> <channel> <embed>
    To make it a embed, type true in embed field.
    Syntax: /automod verify embed <title> <text> <color> <message_id> <icon>

    Removes server's verification system.
    Syntax: /automod verify remove
