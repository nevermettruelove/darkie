Commands

Moderation Commands

/ban <user> <reason>

Bans user.

/kick <user> <reason>

Kicks user.

/banlist

Shows list of banned users.

/mute <user> <time> <reason>

Mutes user.

/unmute <user>

Unmutes user.

/warn <user> <reason>

Warns user.

Information Commands

/server

Sends server information.

/roles

Displays server's role list.

/avatar <user>

Sends user's avatar.

/help

Sends help message.

/ping

Sends current bot's ping.

/user <user>

Sends user information.

Welcome Message Commands

/set welcome message <channel> <message> <embed>

Sets welcome message.

/set welcome embed <title> <color>

Sets welcome embed.

/set leave message <channel> <message> <embed>

Sets leave message.

/set leave embed <title> <color>

Sets leave embed.

Verification System Commands

/setmod verify create <message> <role> <channel> <embed>

Creates verification system.

/setmod verify remove

Removes verification system.

/setmod verify embed <title> <text> <color> <message_id> <icon>

Edits verification system embed.

Autorole Commands

/autorole add <role>

Adds role to autorole.

/autorole remove <role>

Removes role from autorole.

/autorole list

Displays currently added autoroles.
